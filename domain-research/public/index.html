              Beeper drop watch

              Watch a domain's registry status via the free RDAP API and get an email + bell alert the instant it changes — especially the drop to available. For names heading into delete/expiry that are hard to backorder or drop-catch. Adaptive cadence: a name on the cusp is checked every minute; one with an expiration months out is checked occasionally and tightens as the date nears (weekly → daily → hourly → every minute).

                Whois lookup

                A basic, free domain lookup via RDAP (with legacy WHOIS merged in) — registrar, registration / expiry / updated dates, status codes, nameservers, DNSSEC, an email (MX) in-use flag, and the registrant contact when it's public. No credits used.

                DomainIQ ownership history

                Historical WHOIS lineage from DomainIQ — every dated registrant era, distilled to the named owners behind the domain (privacy/registrar layers filtered out). Premium: each lookup spends a DomainIQ credit.
